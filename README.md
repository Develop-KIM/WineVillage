# winevillage
스프링 부트 + JSP 와인 쇼핑몰

## 🖥️ 프로젝트 소개
"와인에 대한 접근성"을 높여 소비자들의 구매력을 **강화**하는, **맞춤형 검색**과 실시간 상담 기능을 갖춘 온라인 와인 쇼핑 플랫폼입니다.

-  `Java 17`
